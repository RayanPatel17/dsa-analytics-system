# DSA Analytics System

A full-stack web application to track and analyze your DSA problem-solving progress.

## Features
- Add, update, and delete problems
- Track problems solved by topic
- Show analytics (total solved, weak topics)
- Smart recommendations for practice
- Revision system for unsolved problems

## Tech Stack
- Python (Flask)
- SQLite
- HTML + JavaScript
